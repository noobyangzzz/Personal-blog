# Blog Vue 3

基于 Vue 3 + Composition API 的博客前端，使用 Vue Router 实现路由导航。

## 技术栈

- Vue 3.4 + Composition API
- Vue Router 4.2
- Vite 5.0

## 快速开始

```bash
npm install
npm run dev    # 开发服务器 http://localhost:3000
npm run build  # 生产构建
```

## 项目结构

```
src/
├── components/    # Vue 组件
├── views/         # 页面视图
├── router/        # 路由配置
├── services/      # API 服务
└── utils/         # 工具函数
```
