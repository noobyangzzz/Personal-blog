## Nginx 配置

已更新 `/home/yang/nginx-1.16.1/conf/blog.conf`:

```nginx
server {
    listen 30000;
    root /home/yang/blog-vue3/dist;
    index index.html index.htm;

    # API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:8080/api/;
    }

    # 静态文章内容
    location /article/ {
        alias /home/yang/blog/article/;
    }

    # Vue Router History 模式支持
    location / {
        try_files $uri $uri/ @fallback;
    }
    location @fallback {
        rewrite ^(.+)$ /index.html last;
    }
}
```

## 渐进式加载实现

文章内容采用渐进式渲染策略，避免大型文章（tech-6.html 约 478KB）阻塞浏览器：

```javascript
// 按 <h1> 标签分割内容
const sections = contentHtml.split(/(?=<h1>)/);

// 使用 requestAnimationFrame 分帧渲染
for (const section of sections) {
    await new Promise(resolve => {
        requestAnimationFrame(() => {
            container.innerHTML += section;
            resolve();
        });
    });
    // 16ms 间隔防止阻塞
    await new Promise(resolve => setTimeout(resolve, 16));
}
```
